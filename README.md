### Tecnologías usadas
 
Desarrollarás un sistema de recomendaciones con base en algoritmos de clustering para Machine Learning, optimizando la ingesta del modelo analizando los principales componentes de los insumos con algoritmos de reducción de la dimensionalidad y procesamiento de lenguaje natural.

* [Python 3.7.13](https://www.python.org/)
